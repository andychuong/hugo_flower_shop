# How to Import into Hugo Manager

This Hugo site is ready to be imported into Hugo Manager. Follow these steps:

## Option 1: Discover Projects
1. Open Hugo Manager
2. Use the "Discover Projects" or "Scan Directory" feature
3. Navigate to the parent directory: `/Users/andychuong/Documents/GauntletAI/Week 8/hugo/`
4. Hugo Manager will automatically detect the `flower-shop` directory as a valid Hugo project

## Option 2: Add Project Manually
1. Open Hugo Manager
2. Click "Add Project" or "Import Project"
3. Enter the project path: `/Users/andychuong/Documents/GauntletAI/Week 8/hugo/flower-shop`
4. Optionally set a custom name (defaults to "flower-shop")

✅ Complete Hugo configuration (`hugo.toml`)

✅ Layout templates (base, home, single, list)
